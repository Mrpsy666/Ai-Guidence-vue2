# 智能导诊系统
### 前后端全栈项目，vue + fastAPI，感兴趣可以自行下载练习，功能逻辑简单，后端下载大模型后便可以启动。

## Frontend Project
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## Backend Project
```
python backendServer/app/main.py
```
